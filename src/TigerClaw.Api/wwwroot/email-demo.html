本页与当前 API 同源，可直接调用 /memory/preferences 与 /workflows/…/run。请先启动：dotnet run --project src/TigerClaw.Api/TigerClaw.Api.csproj，浏览器打开本页地址（见下方）。

为什么会出现 imap.example.com？ 运行时不会自动写这个值。它来自本地 SQLite（data/tigerclaw.db）里已保存的 preferences——例如跑过集成测试、或曾按文档填过示例。CLI 默认用户是 local-user，与下面「用户 Id」一致即可对齐。
演示用主机： 请勿长期使用 imap.example.com（无法解析）。请换成真实 IMAP，如 Gmail：imap.gmail.com，端口 993，并开启应用专用密码等。
